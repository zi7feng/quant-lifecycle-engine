# Epic 3: Order Book and Market Microstructure

## Epic Goal

Add order book support only after the broker data level is confirmed.

MVP only assumes L1 quote data:

```text
best bid
best ask
mid price
spread
spread percentage
```

Do not claim full L2 support until multiple bid / ask price levels are confirmed.
