# Feature 1: Confirm Market Data Level

## PBI E3.1.1: Verify L1 / L2 / LV3 Data

### Checklist

- [ ] Identify available bid / ask fields.
- [ ] Check whether multiple bid levels are available.
- [ ] Check whether multiple ask levels are available.
- [ ] Check whether order-level detail is available.
- [ ] Document the confirmed data level.
- [ ] Update architecture documentation.

### Acceptance Criteria

- [ ] If only best bid and ask are available, document as L1 quote.
- [ ] If multiple bid and ask levels are available, document as L2 market depth.
- [ ] If order-level detail exists, evaluate whether it is LV3.
- [ ] Project documentation does not overclaim data level.
